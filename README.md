# 𝐌𝐨𝐌𝐀.cli 🖼️

### 𝔓𝔯𝔢𝔯𝔢𝔮𝔲𝔦𝔰𝔦𝔱𝔢𝔰

Commands to run project

```

  $ bundle
  
  $ rake db:seed (if you encountered the easter egg)
```

